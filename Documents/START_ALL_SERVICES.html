Sequential Startup (Recommended for First Time)

Open one terminal and follow these steps in order. Each service must start successfully before moving to the next.

1
Start Docker Containers
Navigate to project root and start all services:
cd C:\Projects\vizora\vizora docker-compose up -d
‚úÖ Expected: Messages showing PostgreSQL, Redis, and other services starting. All containers should show "healthy" status.
‚ö†Ô∏è Wait 5-10 seconds for all containers to be ready before proceeding.
2
Start Middleware API
In a new terminal, navigate to middleware:
cd C:\Projects\vizora\vizora\middleware npm run dev
Or using pnpm/nx (alternative):
cd C:\Projects\vizora\vizora pnpm nx serve middleware
‚úÖ Expected: "Listening on port 3000" or "NestJS server started on localhost:3000"
3
Start Realtime WebSocket Server
In a new terminal, navigate to realtime:
cd C:\Projects\vizora\vizora\realtime npm run dev
Or using pnpm/nx (alternative):
cd C:\Projects\vizora\vizora pnpm nx serve realtime
‚úÖ Expected: "Server running on port 3002" or similar WebSocket server message
4
Start Electron Display App
In a new terminal, navigate to display and build:
cd C:\Projects\vizora\vizora\display npm run build
Then clear Node options and start:
unset NODE_OPTIONS && npm start
‚úÖ Expected: Electron window opens showing pairing screen with 6-digit code and QR code

Parallel Startup (Faster - Advanced)

Start services in parallel for faster startup. Open 4 terminals and run these commands simultaneously (after Docker starts).

‚ö†Ô∏è Important: Always start Docker containers FIRST and wait for them to be healthy before starting other services.
1
Start Docker Containers (Sequential - Do This First!)
cd C:\Projects\vizora\vizora docker-compose up -d
‚úÖ Wait for all containers to show "healthy" status (5-10 seconds)
‚Üí
Then open 3 terminals and run these in parallel:
Terminal A: Middleware
cd C:\Projects\vizora\vizora\middleware npm run dev
Terminal B: Realtime
cd C:\Projects\vizora\vizora\realtime npm run dev
Terminal C: Electron
cd C:\Projects\vizora\vizora\display npm run build && unset NODE_OPTIONS && npm start

üìä Troubleshooting Quick Links

Port Already in Use?

Stop and rebuild, or use different ports in docker-compose.yml

Cannot Connect to Database?

Verify containers are running: docker ps

Electron Shows Blank?

Check middleware is on port 3000 with curl http://localhost:3000/api/health

NODE_OPTIONS Error?

Run unset NODE_OPTIONS before starting Electron
